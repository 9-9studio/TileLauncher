# Загрузка TileLauncher

Этот репозиторий предназначен для публичных файлов загрузки TileLauncher. Он не содержит исходный код TileLauncher или код облачных сервисов 9.9 Studio.

## Примечания по загрузке

1. Используйте страницу Releases в этом репозитории, чтобы скачать последнюю стабильную версию.
2. Большинству пользователей нужен установщик `Setup.exe`.
3. `Portable.zip` предназначен для пользователей, которым нужен портативный пакет.
4. Если имя файла, версия или источник выглядят необычно, вернитесь на официальный сайт и заново откройте страницу загрузки.

## Владение

TileLauncher публикуется и поддерживается 9.9 Studio. Этот репозиторий принадлежит организации 9.9 Studio: `9-9studio`.

## Официальные ссылки

- Сайт: https://9.9studio.fun/
- Страница загрузки: https://9.9studio.fun/download/
- Политика конфиденциальности: https://9.9studio.fun/privacy/

## Безопасность

Не скачивайте TileLauncher по неофициальным ссылкам. Официальные файлы создаются процессом выпуска и фиксируются с версией, размером и метаданными SHA-256 в облачной цепочке публикации.
