# Download do TileLauncher

Este repositório hospeda os arquivos públicos de download do TileLauncher. Ele não contém o código-fonte do TileLauncher nem o código dos serviços em nuvem da 9.9 Studio.

## Observações de download

1. Use a página Releases deste repositório para baixar a versão estável mais recente.
2. A maioria dos usuários deve baixar o instalador `Setup.exe`.
3. `Portable.zip` é fornecido para quem prefere um pacote portátil.
4. Se o nome do arquivo, a versão ou a origem parecerem estranhos, volte ao site oficial e abra a página de download novamente.

## Propriedade

TileLauncher é publicado e mantido pela 9.9 Studio. Este repositório pertence à organização 9.9 Studio: `9-9studio`.

## Links oficiais

- Site: https://9.9studio.fun/
- Página de download: https://9.9studio.fun/download/
- Política de privacidade: https://9.9studio.fun/privacy/

## Segurança

Não baixe TileLauncher por links não oficiais. Os arquivos oficiais são gerados pelo processo de release e registrados com versão, tamanho e metadados SHA-256 na cadeia de publicação em nuvem.
