# TileLauncher ダウンロード

このリポジトリは TileLauncher の公開ダウンロードファイルを配布するためのものです。TileLauncher のソースコードや 9.9 Studio のクラウドサービスコードは含まれていません。

## ダウンロードについて

1. 最新の安定版は、このリポジトリの Releases ページからダウンロードしてください。
2. 通常のユーザーは `Setup.exe` インストーラーをダウンロードすれば十分です。
3. `Portable.zip` はポータブル版を使いたいユーザー向けです。
4. ファイル名、バージョン、配布元に違和感がある場合は、公式サイトに戻ってダウンロードページを開き直してください。

## 所有者

TileLauncher は 9.9 Studio が公開し、保守しています。このリポジトリは 9.9 Studio 組織 `9-9studio` が所有しています。

## 公式リンク

- 公式サイト: https://9.9studio.fun/
- ダウンロードページ: https://9.9studio.fun/download/
- プライバシーポリシー: https://9.9studio.fun/privacy/

## 安全について

非公式リンクから TileLauncher をダウンロードしないでください。公式ファイルはリリースプロセスで生成され、バージョン、サイズ、SHA-256 メタデータがクラウドのリリースチェーンに記録されます。
