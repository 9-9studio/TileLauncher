# TileLauncher 다운로드

이 저장소는 TileLauncher의 공개 다운로드 파일을 제공하기 위한 저장소입니다. TileLauncher 소스 코드나 9.9 Studio 클라우드 서비스 코드는 포함하지 않습니다.

## 다운로드 안내

1. 최신 안정 버전은 이 저장소의 Releases 페이지에서 다운로드하세요.
2. 대부분의 사용자는 `Setup.exe` 설치 파일만 받으면 됩니다.
3. `Portable.zip`은 설치 과정을 원하지 않는 사용자를 위한 휴대용 패키지입니다.
4. 파일 이름, 버전 또는 출처가 이상해 보이면 공식 웹사이트로 돌아가 다운로드 페이지를 다시 여세요.

## 소유권

TileLauncher는 9.9 Studio가 배포하고 유지 관리합니다. 이 저장소는 9.9 Studio 조직 `9-9studio`가 소유합니다.

## 공식 링크

- 웹사이트: https://9.9studio.fun/tilelauncher/
- 개인정보 처리방침: https://9.9studio.fun/privacy/

## 보안 안내

비공식 링크에서 TileLauncher를 다운로드하지 마세요. 공식 파일은 릴리스 프로세스를 통해 생성되며, 버전, 크기, SHA-256 메타데이터가 클라우드 릴리스 체인에 기록됩니다.
