# TileLauncher Download

Dieses Repository hostet die öffentlichen Download-Dateien für TileLauncher. Es enthält weder den TileLauncher-Quellcode noch den Cloud-Service-Code von 9.9 Studio.

## Hinweise zum Download

1. Verwenden Sie die Releases-Seite dieses Repositorys für die aktuelle stabile Version.
2. Die meisten Nutzer benötigen nur den `Setup.exe`-Installer.
3. `Portable.zip` ist für Nutzer gedacht, die ein portables Paket bevorzugen.
4. Wenn Dateiname, Version oder Quelle ungewöhnlich wirken, öffnen Sie die Download-Seite erneut über die offizielle Website.

## Eigentum

TileLauncher wird von 9.9 Studio veröffentlicht und gepflegt. Dieses Repository gehört zur 9.9-Studio-Organisation: `9-9studio`.

## Offizielle Links

- Website: https://9.9studio.fun/
- Download-Seite: https://9.9studio.fun/download/
- Datenschutzerklärung: https://9.9studio.fun/privacy/

## Sicherheit

Laden Sie TileLauncher nicht über inoffizielle Links herunter. Offizielle Dateien werden durch den Release-Prozess erzeugt und mit Version, Größe und SHA-256-Metadaten in der Cloud-Release-Kette erfasst.
