# TileLauncher Download

This repository hosts public TileLauncher download files. It does not contain the TileLauncher source code or the 9.9 Studio cloud service code.

## Download Notes

1. Use the Releases page in this repository for the latest stable public download.
2. Most users should download the `Setup.exe` installer.
3. `Portable.zip` is provided for users who prefer a portable package.
4. If the file name, version, or source looks unusual, return to the official website and open the download page again.

## Ownership

TileLauncher is published and maintained by 9.9 Studio. This repository is owned by the 9.9 Studio organization: `9-9studio`.

## Official Links

- Website: https://9.9studio.fun/tilelauncher/
- Privacy policy: https://9.9studio.fun/privacy/

## Safety

Do not download TileLauncher from unofficial links. Official files are generated by the release process and recorded with version, size, and SHA-256 metadata in the cloud release chain.
