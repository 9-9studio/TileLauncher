# Descarga de TileLauncher

Este repositorio aloja los archivos públicos de descarga de TileLauncher. No contiene el código fuente de TileLauncher ni el código de servicios en la nube de 9.9 Studio.

## Notas de descarga

1. Use la página Releases de este repositorio para descargar la versión estable más reciente.
2. La mayoría de usuarios solo necesita el instalador `Setup.exe`.
3. `Portable.zip` está disponible para quienes prefieren un paquete portable.
4. Si el nombre del archivo, la versión o la fuente parecen extraños, vuelva al sitio web oficial y abra de nuevo la página de descarga.

## Propiedad

TileLauncher es publicado y mantenido por 9.9 Studio. Este repositorio pertenece a la organización 9.9 Studio: `9-9studio`.

## Enlaces oficiales

- Sitio web: https://9.9studio.fun/
- Página de descarga: https://9.9studio.fun/download/
- Política de privacidad: https://9.9studio.fun/privacy/

## Seguridad

No descargue TileLauncher desde enlaces no oficiales. Los archivos oficiales se generan mediante el proceso de publicación y se registran con versión, tamaño y metadatos SHA-256 en la cadena de publicación en la nube.
