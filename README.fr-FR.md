# Téléchargement de TileLauncher

Ce dépôt héberge les fichiers publics de téléchargement de TileLauncher. Il ne contient pas le code source de TileLauncher ni le code des services cloud de 9.9 Studio.

## Notes de téléchargement

1. Utilisez la page Releases de ce dépôt pour obtenir la dernière version stable.
2. La plupart des utilisateurs doivent télécharger l'installateur `Setup.exe`.
3. `Portable.zip` est fourni pour les utilisateurs qui préfèrent un paquet portable.
4. Si le nom du fichier, la version ou la source semble inhabituel, revenez au site officiel et ouvrez de nouveau la page de téléchargement.

## Propriété

TileLauncher est publié et maintenu par 9.9 Studio. Ce dépôt appartient à l'organisation 9.9 Studio : `9-9studio`.

## Liens officiels

- Site web : https://9.9studio.fun/tilelauncher/
- Politique de confidentialité : https://9.9studio.fun/privacy/

## Sécurité

Ne téléchargez pas TileLauncher depuis des liens non officiels. Les fichiers officiels sont générés par le processus de publication et enregistrés avec leur version, leur taille et leurs métadonnées SHA-256 dans la chaîne de publication cloud.
