# TileLauncher 下載

本倉庫用於發布 TileLauncher 的公開下載檔案。它不承載 TileLauncher 原始碼，也不承載 9.9 Studio 雲服務程式碼。

## 下載須知

1. 請優先從本倉庫的 Releases 頁面下載最新穩定版。
2. 一般使用者通常只需要下載 `Setup.exe` 安裝檔。
3. `Portable.zip` 是可攜版，適合不想執行安裝流程的使用者。
4. 如果檔名、版本號或來源看起來異常，請回到官網重新進入下載頁。

## 所有權

TileLauncher 由 9.9 Studio 發布和維護。本倉庫歸屬於 9.9 Studio 組織：`9-9studio`。

## 官方連結

- 官網：https://9.9studio.fun/tilelauncher/
- 隱私政策：https://9.9studio.fun/privacy/

## 安全說明

請不要從非官方連結下載 TileLauncher。正式下載檔案會透過發布流程生成，並在雲端發布鏈路中記錄版本、大小和 SHA-256 校驗資訊。
