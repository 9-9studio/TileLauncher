# Pobieranie TileLauncher

To repozytorium przechowuje publiczne pliki pobierania TileLauncher. Nie zawiera kodu źródłowego TileLauncher ani kodu usług chmurowych 9.9 Studio.

## Informacje o pobieraniu

1. Najnowszą stabilną wersję pobieraj ze strony Releases w tym repozytorium.
2. Większość użytkowników powinna pobrać instalator `Setup.exe`.
3. `Portable.zip` jest przeznaczony dla użytkowników preferujących pakiet przenośny.
4. Jeśli nazwa pliku, wersja lub źródło wyglądają nietypowo, wróć na oficjalną stronę i ponownie otwórz stronę pobierania.

## Własność

TileLauncher jest publikowany i utrzymywany przez 9.9 Studio. To repozytorium należy do organizacji 9.9 Studio: `9-9studio`.

## Oficjalne linki

- Strona: https://9.9studio.fun/
- Pobieranie: https://9.9studio.fun/download/
- Polityka prywatności: https://9.9studio.fun/privacy/

## Bezpieczeństwo

Nie pobieraj TileLauncher z nieoficjalnych linków. Oficjalne pliki są generowane przez proces wydania i zapisywane z metadanymi wersji, rozmiaru oraz SHA-256 w chmurowym łańcuchu wydań.
