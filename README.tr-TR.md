# TileLauncher İndirme

Bu depo, TileLauncher için herkese açık indirme dosyalarını barındırır. TileLauncher kaynak kodunu veya 9.9 Studio bulut hizmeti kodunu içermez.

## İndirme Notları

1. En son kararlı sürümü indirmek için bu deponun Releases sayfasını kullanın.
2. Çoğu kullanıcı yalnızca `Setup.exe` kurulum dosyasını indirmelidir.
3. `Portable.zip`, taşınabilir paket isteyen kullanıcılar için sağlanır.
4. Dosya adı, sürüm veya kaynak olağan dışı görünüyorsa resmi siteye dönün ve indirme sayfasını yeniden açın.

## Sahiplik

TileLauncher, 9.9 Studio tarafından yayımlanır ve bakım altında tutulur. Bu depo 9.9 Studio kuruluşuna aittir: `9-9studio`.

## Resmi Bağlantılar

- Web sitesi: https://9.9studio.fun/tilelauncher/
- Gizlilik politikası: https://9.9studio.fun/privacy/

## Güvenlik

TileLauncher'ı resmi olmayan bağlantılardan indirmeyin. Resmi dosyalar yayın süreciyle oluşturulur ve bulut yayın zincirinde sürüm, boyut ve SHA-256 meta verileriyle kaydedilir.
