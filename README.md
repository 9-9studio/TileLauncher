# TileLauncher 下载

本仓库用于发布 TileLauncher 的公开下载文件。它不承载 TileLauncher 源代码，也不承载 9.9 Studio 云服务代码。

## 下载须知

1. 请优先从本仓库的 Releases 页面下载最新稳定版。
2. 普通用户通常只需要下载 `Setup.exe` 安装包。
3. `Portable.zip` 是便携包，适合不希望运行安装流程的用户。
4. 如果文件名、版本号或来源看起来异常，请回到官网重新进入下载页。

## 所有权

TileLauncher 由 9.9 Studio 发布和维护。本仓库归属于 9.9 Studio 组织：`9-9studio`。

## 官方链接

- 官网：https://9.9studio.fun/tilelauncher/
- 隐私政策：https://9.9studio.fun/privacy/

## 安全说明

请不要从非官方链接下载 TileLauncher。正式下载文件会通过发布流程生成，并在云端发布链路中记录版本、大小和 SHA-256 校验信息。
